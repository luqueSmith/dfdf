Pregunta 01

Servicios centralizados de gestión de autenticación, para los usuarios de acceso remoto:

  • check_small RADIUS.

  • close_small DHCP.

  • close_small BOOTP.

  • close_small OSPF.

  • close_small EIGRP.

Respuesta

RADIUS (Remote Authentication Dial-In User Service).RADIUS es un protocolo utilizado para la autenticación, autorización y contabilidad de usuarios que se conectan a una red, especialmente en el acceso remoto y redes inalámbricas.

Pregunta 02

En una red de datos, puede utilizar el servicio de RADIUS para autenticarse en:

  • check_small Switches.

  • close_small Carpetas.

  • close_small Dominio.

  • close_small Impresora.

  • close_small Database.

Respuesta

Switches RADIUS puede ser utilizado para autenticar y autorizar el acceso a dispositivos de red como switches, routers y puntos de acceso, proporcionando un control centralizado sobre los usuarios que intentan acceder a estos dispositivos.

Pregunta 03

Servicio que permite configurar y administrar la autenticación y autorización RADIUS:

  • check_small NPS.

  • close_small SSL.

  • close_small SSH.

  • close_small VPN.

  • close_small RTP.

Respuesta

NPS (Network Policy Server).NPS es el servicio de Windows Server que implementa RADIUS y se utiliza para gestionar políticas de red, autenticación, autorización y contabilidad para usuarios que se conectan a la red.

Pregunta 04

El protocolo RADIUS utiliza un esquema de tipo:

  • check_small Cliente servidor.

  • close_small Punto a redes.

  • close_small Maestro esclavo.

  • close_small Servicio local.

  • close_small Base de datos.

Respuesta

Cliente servidor. En este esquema, un cliente RADIUS (como un switch o un punto de acceso) envía solicitudes de autenticación a un servidor RADIUS, que verifica las credenciales del usuario y proporciona una respuesta sobre si el acceso debe ser concedido o denegado.

Pregunta 05

Protocolo que usan los proveedores de internet con RADIUS:

  • check_small PPoE.

  • close_small RAS.

  • close_small WiFi.

  • close_small SSH.

  • close_small SSL.

Respuesta

PPPoE (Point-to-Point Protocol over Ethernet).PPPoE se utiliza para encapsular paquetes de datos en redes Ethernet y es común en las conexiones de banda ancha de los proveedores de Internet. RADIUS se emplea para autenticar y autorizar usuarios que se conectan a través de PPPoE.
