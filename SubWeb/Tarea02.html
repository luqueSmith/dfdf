RESPUESTAS AUTOEVALUACION 02 Regresar

Pregunta 01

Una configuración opcional enviada por el servidor DHCP es información de:

  • check_small DNS.

  • close_small IP.

  • close_small FTP.

  • close_small MAC.

  • close_small SSL.

Respuesta

DNS (Domain Name System). Además de asignar direcciones IP, un servidor DHCP puede enviar información adicional como la dirección de los servidores DNS, la puerta de enlace predeterminada (gateway) y las máscaras de subred.

Pregunta 02

A la suplantación de DHCP se le conoce como:

  • check_small Spoofing.

  • close_small Snooping.

  • close_small Adware.

  • close_small Spyware.

  • close_small Malware.

Respuesta

Spoofing. El DHCP spoofing es un tipo de ataque en el que un atacante configura un servidor DHCP no autorizado en la red para distribuir direcciones IP falsas u otra información incorrecta a los clientes DHCP, con el fin de interceptar o manipular el tráfico de red.

Pregunta 03

Las IPS de equipos de red se configuran en DHCP como direcciones:

  • check_small Excluidas.

  • close_small Externas.

  • close_small Válidas.

  • close_small Grupales.

  • close_small Cedidas.

Respuesta

Excluidas. En DHCP, las direcciones excluidas son aquellas que se configuran para no ser asignadas automáticamente por el servidor DHCP. Estas direcciones suelen ser utilizadas para dispositivos de red como routers, switches, servidores y otros equipos que requieren una dirección IP estática.

Pregunta 04

Sin DHCP las computadoras de una red deben de configurar sus direcciones IP de forma:

  • check_small Estática.

  • close_small Aleatoria.

  • close_small Dinámica.

  • close_small Semanal.

  • close_small Ocasional.

Respuesta

Estática. En ausencia de un servidor DHCP, cada dispositivo en la red debe tener su dirección IP configurada manualmente (estática) para asegurarse de que no haya conflictos y que cada dispositivo pueda comunicarse correctamente en la red.

Pregunta 05

Si un router asigna direcciones IP que no se deberían asignar, esto se evita mediante:

  • check_small Snooping.

  • close_small Antivirus.

  • close_small Firewall.

  • close_small Directivas.

  • close_small Spoofing.

Respuesta

Snooping. El DHCP snooping es una técnica utilizada para proteger la red contra servidores DHCP no autorizados (maliciosos) que intentan asignar direcciones IP incorrectas. Esta técnica permite que los switches supervisen las comunicaciones DHCP y bloqueen cualquier respuesta de servidores DHCP no autorizados.
