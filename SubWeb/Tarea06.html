Pregunta 01

Cada usuario de Exchange puede tener como máximo de buzones:

  • check_small 1.

  • close_small 4.

  • close_small 5.

  • close_small 2.

  • close_small 3.

Respuesta

1 buzón.En Microsoft Exchange, cada usuario tiene un buzón principal asociado con su cuenta. Sin embargo, los usuarios pueden tener buzones adicionales, como buzones compartidos o buzones de recurso, que no cuentan como buzones personales.

Pregunta 02

Para crear un nuevo buzón de correo se utiliza:

  • check_small Exchange admin center.

  • close_small Active directory for EAC.

  • close_small Exchange active directory.

  • close_small Linked mailbox Exchange.

  • close_small Mailbox Exchange center.

Respuesta

Exchange Admin Center (EAC).El Exchange Admin Center es la interfaz gráfica de administración basada en web que permite a los administradores gestionar buzones de correo, configuraciones de Exchange y otros aspectos relacionados con el correo electrónico.

  • check_small Usuarios.

  • close_small Exchange.

  • close_small Mailbox.

Respuesta

Usuarios.En Active Directory, por defecto, los nuevos usuarios se crean en el contenedor "Usuarios", a menos que se especifique una ubicación diferente durante el proceso de creación.
